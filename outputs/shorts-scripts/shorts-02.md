# 쇼츠 2: Loop 닫기 - AI 코딩의 핵심 원칙

## 타임라인

### [0:00-0:03] 후킹
당신의 AI는 자신이 작성한 코드를 테스트할 수 있나요?

### [0:03-0:50] 본문
많은 사람들이 AI에게 코드를 짜라고 하는데, 여기서 끝나요.

하지만 Peter Steinberger가 말하는 "Loop 닫기"는 달라요.

AI가 코드를 짠 후, 스스로 테스트하고, 버그를 찾고, 고쳐야 한다는 거예요.

이렇게 하면 AI가 자동으로 검증된 코드를 만들어요.

시스템을 이렇게 설계하면, 당신은 나중에 코드를 읽을 필요가 없어요.

- (자막: Loop 닫기, 자동 테스트, 검증)
- (화면: 코드 → 테스트 → 수정 → 배포 다이어그램)
- (배경음: 안정적이고 신뢰할 수 있는 느낌)

### [0:50-1:00] CTA
개발 방식이 바뀔 거예요. 저장해두고 나중에 써먹으세요!
