# 카드뉴스 제작 가이드

인스타그램 카드뉴스 제작을 위한 상세 가이드입니다.

## 1. 기본 스펙

### 이미지 사양

| 항목 | 값 |
|------|-----|
| 크기 | 1080 x 1080px (정사각형) |
| 비율 | 1:1 |
| 형식 | PNG (권장) / JPG |
| 해상도 | 72dpi (웹용) |
| 컬러 | sRGB |

### 슬라이드 수

- 최소: 5장
- 최대: 10장 (인스타그램 제한)
- 권장: 5-7장

## 2. 슬라이드별 구성

### 슬라이드 1: 표지

**목적**: 스크롤을 멈추게 하는 첫인상

```
구성요소:
├── 메인 제목 (후킹)
├── 서브 제목 (선택)
├── 브랜드 로고 (선택)
└── 시선을 끄는 비주얼

텍스트 예시:
✅ "이거 모르면 손해예요"
✅ "3가지만 기억하세요"
✅ "1분 만에 이해하는 [주제]"

❌ "[주제]에 대해 알아보기"
❌ "안녕하세요, 오늘은..."
```

### 슬라이드 2-6: 본문

**목적**: 핵심 정보 전달

```
구성요소:
├── 슬라이드 번호 (1, 2, 3...)
├── 소제목
├── 핵심 내용 (2-3줄)
├── 아이콘/일러스트 (선택)
└── 하이라이트 박스 (선택)

원칙:
- 한 슬라이드 = 한 메시지
- 텍스트는 50단어 이내
- 핵심 키워드 강조
```

### 마지막 슬라이드: CTA

**목적**: 행동 유도

```
구성요소:
├── 요약 또는 핵심 메시지
├── CTA 문구
├── 팔로우/저장 유도
└── 브랜드 정보

CTA 예시:
✅ "저장해두고 나중에 보세요 📌"
✅ "도움이 됐다면 공유해주세요 💙"
✅ "더 많은 팁은 팔로우! 👆"
```

## 3. 디자인 원칙

### 레이아웃

```
┌─────────────────────────────┐
│         여백 (상)            │
│  ┌─────────────────────┐   │
│  │                     │   │
│  │    콘텐츠 영역       │   │
│  │    (80% 이내)       │   │
│  │                     │   │
│  └─────────────────────┘   │
│         여백 (하)            │
└─────────────────────────────┘

- 상하좌우 여백: 최소 50px
- 텍스트 영역: 화면의 80% 이내
- 중요 요소: 중앙 또는 황금비율
```

### 텍스트 계층

```
제목 (H1): 36-48px, Bold
소제목 (H2): 24-32px, SemiBold
본문: 18-24px, Regular
캡션: 14-16px, Light

줄간격: 1.4-1.6
자간: 0 ~ -25
```

### 색상 사용

```
배경색:
- 밝은 톤 권장 (가독성)
- 브랜드 컬러 10-20% 사용

텍스트 색:
- 배경과 명확한 대비
- 다크모드: #1F2937
- 라이트모드: #F9FAFB

강조색:
- 브랜드 컬러 사용
- 한 슬라이드에 1-2색 제한
```

## 4. 콘텐츠 작성 가이드

### 텍스트 작성

```
✅ 좋은 예:
- 짧고 명확한 문장
- 구체적인 숫자 사용
- 행동 지향적 표현
- "~해요" 체 사용

❌ 피해야 할 예:
- 긴 문장
- 추상적인 표현
- 전문 용어 (설명 없이)
- "~합니다" 체 (딱딱함)
```

### 번호 매기기

```
방법 1: 숫자
1, 2, 3, 4, 5

방법 2: 이모지
✅, 📌, 💡, ⭐, 🎯

방법 3: 원문자
①, ②, ③, ④, ⑤
```

## 5. 프롬프트 작성 팁

### 나노바나나용 프롬프트 구조

```
[스타일] + [레이아웃] + [색상] + [텍스트 내용] + [분위기]

예시:
"미니멀한 카드뉴스 슬라이드,
흰색 배경에 파란색 포인트,
중앙에 큰 숫자 '1',
아래에 '첫 번째 팁' 텍스트,
깔끔하고 전문적인 느낌"
```

### 일관성 유지

```
모든 슬라이드에 동일하게:
- 배경 스타일
- 폰트 스타일
- 색상 팔레트
- 여백 크기
- 로고 위치
```

## 6. 체크리스트

### 제작 전

```
□ 브리프에서 핵심 메시지 확인
□ 슬라이드 수 결정 (5-7장)
□ 각 슬라이드 내용 정리
□ 스타일/무드 결정
```

### 제작 후

```
□ 모든 텍스트 오타 확인
□ 색상 대비 확인 (가독성)
□ 슬라이드 순서 논리적
□ CTA 명확함
□ 브랜드 일관성 유지
□ 파일 크기 적정 (슬라이드당 1MB 이내)
```
