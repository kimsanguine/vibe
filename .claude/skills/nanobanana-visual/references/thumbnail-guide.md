# 유튜브 썸네일 제작 가이드

클릭을 유도하는 효과적인 썸네일 제작을 위한 상세 가이드입니다.

## 1. 기본 스펙

### 이미지 사양

| 항목 | 값 |
|------|-----|
| 크기 | 1280 x 720px |
| 비율 | 16:9 |
| 형식 | PNG (권장) / JPG |
| 최대 용량 | 2MB |
| 해상도 | 72dpi |

### 안전 영역

```
┌─────────────────────────────────────┐
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │      안전 영역 (90%)         │   │
│  │      핵심 콘텐츠 배치        │   │
│  │                             │   │
│  │                     ┌─────┐ │   │
│  │                     │시간 │ │   │
│  └─────────────────────┴─────┴─┘   │
└─────────────────────────────────────┘

- 오른쪽 하단: 영상 길이 표시 영역 (피하기)
- 가장자리 5%: 잘릴 수 있음
- 핵심 요소는 중앙 90%에 배치
```

## 2. 썸네일 구성 요소

### 필수 요소

```
1. 메인 텍스트
   - 영상의 핵심 키워드
   - 3-6단어 권장
   - 화면의 30-40% 차지

2. 배경/이미지
   - 주제를 나타내는 비주얼
   - 단순하고 명확하게
```

### 선택 요소

```
3. 얼굴/인물
   - 감정이 드러나는 표정
   - CTR 향상에 효과적

4. 아이콘/그래픽
   - 화살표, 동그라미, 강조
   - 시선 유도에 활용

5. 브랜드 요소
   - 로고 (작게)
   - 채널 컬러
```

## 3. 디자인 원칙

### 텍스트 가이드

```
크기:
- 메인 텍스트: 화면의 30-40%
- 서브 텍스트: 메인의 50-60%

위치:
- 중앙 또는 좌측 상단
- 얼굴이 있으면 반대편

스타일:
- Bold 또는 Extra Bold
- 테두리/그림자로 가독성 확보
- 최대 2줄

✅ 좋은 예:
"이거 모르면 손해"
"30% 절약하는 방법"
"초보자 필수 가이드"

❌ 피해야 할 예:
"오늘 영상에서는 AI 자동화 도구에 대해서 알아보겠습니다"
(너무 김)
```

### 색상 원칙

```
대비:
- 배경과 텍스트 강한 대비
- 어두운 배경 + 밝은 텍스트
- 밝은 배경 + 어두운 텍스트

색상 조합 예시:
1. 파랑 배경 + 노랑 텍스트
2. 빨강 배경 + 흰색 텍스트
3. 검정 배경 + 노랑 텍스트
4. 흰색 배경 + 빨강/파랑 텍스트

브랜드 컬러:
- 일관되게 사용
- 채널 인식도 향상
```

### 감정 표현

```
효과적인 표정:
😲 놀람 - "이게 뭐야?"
😄 기쁨 - "이거 대박!"
🤔 궁금함 - "어떻게 된 거지?"
😱 충격 - "말도 안 돼"

표정이 없는 경우:
- 화살표로 시선 유도
- 물음표/느낌표 활용
- 이모지 사용
```

## 4. 클릭을 유도하는 요소

### 심리적 트리거

```
1. 호기심 자극
   - "아무도 안 알려주는"
   - "숨겨진 비밀"
   - "이것만 알면"

2. 긴급성
   - "지금 바로"
   - "오늘만"
   - "놓치면 후회"

3. 숫자 활용
   - "5가지 방법"
   - "30% 절약"
   - "10분 만에"

4. 비교/대조
   - "전 vs 후"
   - "A vs B"
   - "초보 vs 고수"
```

### 시선 유도 기법

```
1. 화살표
   → ↓ ↗ (방향 강조)

2. 동그라미
   ○ (특정 부분 강조)

3. 선/밑줄
   ___ (텍스트 강조)

4. 프레임
   [ ] (영역 강조)
```

## 5. 프롬프트 작성

### 나노바나나용 프롬프트 구조

```
[배경 설명] + [인물/요소] + [텍스트 위치] + [색상] + [분위기]

예시:
"유튜브 썸네일,
그라데이션 파란색 배경,
왼쪽에 놀란 표정의 남성,
오른쪽에 'AI로 월 100만원' 노란색 굵은 텍스트,
역동적이고 눈에 띄는 스타일"
```

### 프롬프트 체크리스트

```
□ 배경 스타일 명시
□ 인물/요소 설명
□ 텍스트 내용과 위치
□ 색상 지정
□ 전체 분위기
□ 16:9 비율 명시
```

## 7. 체크리스트

### 제작 전

```
□ 영상 핵심 키워드 정리
□ 경쟁 영상 썸네일 분석
□ 레이아웃 스케치
□ 색상 팔레트 결정
```

### 제작 후

```
□ 모바일에서 확인 (작게 봐도 읽히는지)
□ 텍스트 가독성 확인
□ 색상 대비 확인
□ 우하단 시간 표시 영역 비움
□ 파일 크기 2MB 이내
□ 채널 다른 썸네일과 일관성
```

### A/B 테스트 포인트

```
테스트 가능한 요소:
- 텍스트 문구
- 색상 조합
- 얼굴 표정
- 레이아웃

24-48시간 후 CTR 비교
```

## 8. 자주 하는 실수

```
❌ 텍스트가 너무 작음
   → 모바일에서 안 보임

❌ 색상 대비 부족
   → 텍스트가 배경에 묻힘

❌ 요소가 너무 많음
   → 시선 분산

❌ 영상 제목 그대로 복사
   → 중복, 지루함

❌ 저화질 이미지
   → 전문성 떨어짐

❌ 우하단에 중요 요소
   → 시간 표시에 가려짐
```
