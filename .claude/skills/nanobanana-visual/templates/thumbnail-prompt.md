# 썸네일 프롬프트 템플릿

나노바나나 3.0 프로에서 유튜브 썸네일을 생성하기 위한 프롬프트 템플릿입니다.

## 기본 프롬프트 구조

```
[형식] + [배경] + [인물/요소] + [텍스트] + [색상] + [분위기]
```

## 레이아웃별 템플릿

### Type A: 얼굴 + 텍스트

```markdown
## 프롬프트

유튜브 썸네일, 1280x720, 16:9 비율,
{{BACKGROUND_STYLE}} 배경,
왼쪽에 {{PERSON_DESCRIPTION}} 인물 ({{EXPRESSION}} 표정),
오른쪽에 "{{MAIN_TEXT}}" {{TEXT_COLOR}} 굵은 텍스트,
텍스트에 {{TEXT_EFFECT}} 효과,
{{MOOD}} 분위기

## 변수

- BACKGROUND_STYLE: 배경 스타일 (예: 그라데이션 파란색)
- PERSON_DESCRIPTION: 인물 설명 (예: 30대 남성, 정장)
- EXPRESSION: 표정 (예: 놀란, 기쁜, 진지한)
- MAIN_TEXT: 메인 텍스트 (3-6단어)
- TEXT_COLOR: 텍스트 색상 (예: 노란색)
- TEXT_EFFECT: 텍스트 효과 (예: 테두리, 그림자)
- MOOD: 전체 분위기 (예: 역동적인, 전문적인)
```

### Type B: 텍스트 중심

```markdown
## 프롬프트

유튜브 썸네일, 1280x720, 16:9 비율,
{{BACKGROUND_STYLE}} 배경,
중앙에 "{{MAIN_TEXT}}" 매우 큰 {{TEXT_COLOR}} 텍스트,
아래에 "{{SUB_TEXT}}" 작은 텍스트,
{{GRAPHIC_ELEMENT}} 그래픽 요소,
텍스트에 {{TEXT_EFFECT}} 효과,
{{MOOD}} 분위기

## 변수

- BACKGROUND_STYLE: 배경 스타일
- MAIN_TEXT: 메인 텍스트 (핵심 키워드)
- SUB_TEXT: 서브 텍스트 (보조 정보)
- TEXT_COLOR: 메인 텍스트 색상
- GRAPHIC_ELEMENT: 장식 요소 (예: 화살표, 아이콘)
- TEXT_EFFECT: 텍스트 효과
- MOOD: 전체 분위기
```

### Type C: 비교형

```markdown
## 프롬프트

유튜브 썸네일, 1280x720, 16:9 비율,
화면 중앙에 "VS" 텍스트,
왼쪽에 "{{LEFT_ITEM}}" ({{LEFT_COLOR}} 톤),
오른쪽에 "{{RIGHT_ITEM}}" ({{RIGHT_COLOR}} 톤),
대비되는 색상 구성,
상단 또는 하단에 "{{QUESTION_TEXT}}" 텍스트,
경쟁적이고 흥미로운 분위기

## 변수

- LEFT_ITEM: 왼쪽 비교 대상
- LEFT_COLOR: 왼쪽 영역 색상
- RIGHT_ITEM: 오른쪽 비교 대상
- RIGHT_COLOR: 오른쪽 영역 색상
- QUESTION_TEXT: 질문 텍스트 (예: "어떤 게 더 좋을까?")
```

### Type D: 숫자 강조형

```markdown
## 프롬프트

유튜브 썸네일, 1280x720, 16:9 비율,
{{BACKGROUND_STYLE}} 배경,
중앙에 매우 큰 숫자 "{{NUMBER}}" ({{NUMBER_STYLE}}),
숫자 옆에 "{{UNIT_TEXT}}" 텍스트,
하단에 "{{DESCRIPTION}}" 설명,
{{ACCENT_ELEMENT}} 강조 요소,
임팩트 있고 눈에 띄는 스타일

## 변수

- NUMBER: 강조할 숫자 (예: 30%, 5가지, 100만)
- NUMBER_STYLE: 숫자 스타일 (예: 3D, 네온)
- UNIT_TEXT: 단위 또는 설명 (예: % 절약, 가지 방법)
- DESCRIPTION: 추가 설명
- ACCENT_ELEMENT: 강조 요소 (예: 별, 화살표)
```

## 감정별 프롬프트 예시

### 놀람/충격

```
유튜브 썸네일,
밝은 노란색 또는 빨간색 배경,
"😱" 또는 놀란 표정의 인물,
"말도 안 돼!" 또는 "이게 가능해?" 텍스트,
느낌표 강조,
충격적이고 긴급한 분위기
```

### 정보/교육

```
유튜브 썸네일,
깔끔한 파란색 또는 흰색 배경,
진지하거나 설명하는 표정의 인물,
"완벽 가이드" 또는 "꿀팁" 텍스트,
체크마크 또는 책 아이콘,
전문적이고 신뢰감 있는 분위기
```

### 성공/결과

```
유튜브 썸네일,
밝은 초록색 또는 금색 포인트,
기쁜 표정 또는 성취 포즈,
숫자 강조 (수익, 성장률),
상승 화살표 그래픽,
성공적이고 긍정적인 분위기
```

```markdown

### 메인 텍스트
{{BRIEF.THUMBNAIL_TEXT}}

### 스타일 가이드
- 색상: {{BRIEF.THUMBNAIL_COLOR}}
- 분위기: {{BRIEF.THUMBNAIL_MOOD}}

### 타겟 오디언스
{{BRIEF.TARGET_AUDIENCE}}
```

## 색상 조합 참고

```
고클릭률 색상 조합:

1. 파랑 + 노랑
   - 배경: #1E40AF (진한 파랑)
   - 텍스트: #FCD34D (노랑)

2. 빨강 + 흰색
   - 배경: #DC2626 (빨강)
   - 텍스트: #FFFFFF (흰색)

3. 검정 + 노랑
   - 배경: #1F2937 (검정)
   - 텍스트: #FCD34D (노랑)

4. 보라 + 흰색
   - 배경: #7C3AED (보라)
   - 텍스트: #FFFFFF (흰색)
```

## 체크리스트

프롬프트 작성 시 확인:

```
□ 이미지 크기 명시 (1280x720, 16:9)
□ 배경 스타일 명확히
□ 메인 텍스트 내용 포함 (3-6단어)
□ 텍스트 색상 및 효과 지정
□ 인물 있는 경우 표정 명시
□ 전체 분위기 설명
□ 우하단 영역 비워두기 언급
```

## 금지 요소

```
프롬프트에서 피해야 할 것:

❌ 저작권 있는 캐릭터/로고
❌ 실제 유명인 얼굴
❌ 폭력적/선정적 요소
❌ 잘못된 정보/클릭베이트
❌ 너무 복잡한 레이아웃
```
