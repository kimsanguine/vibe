# 카드뉴스 프롬프트 템플릿

나노바나나 3.0 프로에서 카드뉴스를 생성하기 위한 프롬프트 템플릿입니다.

## 기본 프롬프트 구조

```
[콘텐츠 유형] + [레이아웃] + [색상] + [텍스트] + [스타일]
```

## 슬라이드별 템플릿

### 슬라이드 1: 표지

```markdown
## 프롬프트

인스타그램 카드뉴스 표지 슬라이드,
1080x1080 정사각형,
{{BACKGROUND_COLOR}} 배경에 {{ACCENT_COLOR}} 포인트,
중앙에 "{{MAIN_TITLE}}" 큰 텍스트 ({{FONT_STYLE}}),
아래에 "{{SUB_TITLE}}" 작은 텍스트,
{{STYLE_MOOD}} 스타일,
깔끔하고 전문적인 느낌

## 변수

- BACKGROUND_COLOR: 배경색 (예: 흰색, 연한 파란색)
- ACCENT_COLOR: 강조색 (예: 진한 파란색)
- MAIN_TITLE: 메인 제목 (예: "이거 모르면 손해")
- SUB_TITLE: 부제목 (예: "AI 자동화 꿀팁 5가지")
- FONT_STYLE: 폰트 스타일 (예: Bold, Modern)
- STYLE_MOOD: 전체 분위기 (예: 미니멀, 모던, 따뜻한)
```

### 슬라이드 2-6: 본문

```markdown
## 프롬프트

인스타그램 카드뉴스 본문 슬라이드,
1080x1080 정사각형,
{{BACKGROUND_COLOR}} 배경,
좌측 상단에 숫자 "{{NUMBER}}" ({{NUMBER_STYLE}}),
중앙에 "{{POINT_TITLE}}" 소제목,
아래에 "{{POINT_CONTENT}}" 본문 텍스트,
{{ICON_ELEMENT}} 아이콘 포함,
이전 슬라이드와 동일한 스타일 유지

## 변수

- NUMBER: 슬라이드 번호 (1, 2, 3...)
- NUMBER_STYLE: 숫자 스타일 (예: 원형 배경, 큰 숫자)
- POINT_TITLE: 포인트 제목
- POINT_CONTENT: 포인트 내용 (2-3줄)
- ICON_ELEMENT: 관련 아이콘 설명
```

### 마지막 슬라이드: CTA

```markdown
## 프롬프트

인스타그램 카드뉴스 마지막 슬라이드,
1080x1080 정사각형,
{{BACKGROUND_COLOR}} 배경,
중앙에 "{{CTA_TEXT}}" 텍스트,
{{CTA_ICON}} 아이콘,
"저장 & 공유" 유도 문구,
브랜드 로고 또는 계정명 하단에,
이전 슬라이드와 동일한 스타일

## 변수

- CTA_TEXT: CTA 문구 (예: "도움이 됐다면 저장!")
- CTA_ICON: CTA 아이콘 (예: 북마크, 하트, 공유)
```

## 스타일별 프롬프트 예시

### 미니멀 스타일

```
인스타그램 카드뉴스,
미니멀 디자인,
흰색 배경에 검정 텍스트,
파란색 포인트 컬러,
충분한 여백,
깔끔한 산세리프 폰트,
전문적이고 현대적인 느낌
```

### 그라데이션 스타일

인스타그램 카드뉴스,

흰색 텍스트,
부드러운 곡선 요소,
글로우 효과,
트렌디하고 역동적인 느낌
```

둥근 폰트,

캐주얼한 스타일
```

## 일관성 유지를 위한 공통 프롬프트

```markdown
## 공통 스타일 지시

모든 슬라이드에 적용:
- 동일한 배경 스타일: {{BACKGROUND_STYLE}}
- 동일한 폰트 패밀리: {{FONT_FAMILY}}
- 동일한 컬러 팔레트: {{COLOR_PALETTE}}
- 동일한 여백: 상하좌우 50px
- 동일한 텍스트 정렬: {{ALIGNMENT}}
```

## 브리프 연동 프롬프트

```markdown
## brief.md 참조 프롬프트

다음 브리프 정보를 반영하여 카드뉴스 생성:

### 핵심 메시지
{{BRIEF.KEY_MESSAGES}}

### 비주얼 가이드
- 슬라이드 수: {{BRIEF.SLIDE_COUNT}}
- 핵심 비주얼: {{BRIEF.VISUAL_CONCEPT}}
- 텍스트 키워드: {{BRIEF.TEXT_KEYWORDS}}

### 톤앤매너
- 스타일: {{BRIEF.STYLE}}
- 타겟: {{BRIEF.TARGET_AUDIENCE}}
```

## 체크리스트

프롬프트 작성 시 확인:

```
□ 이미지 크기 명시 (1080x1080)
□ 배경 색상/스타일 명시
□ 텍스트 내용 포함
□ 폰트 스타일 지정
□ 전체 분위기 설명
□ 이전 슬라이드와 일관성 언급
```
