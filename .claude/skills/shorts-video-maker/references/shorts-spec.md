# 쇼츠 영상 스펙 가이드

YouTube Shorts, Instagram Reels, TikTok 등 숏폼 플랫폼의 영상 스펙입니다.

## 1. 공통 스펙

### 영상 비율

| 항목 | 값 |
|------|-----|
| 비율 | 9:16 (세로) |
| 해상도 | 1080 x 1920px |
| 대체 해상도 | 720 x 1280px (최소) |

### 길이

| 플랫폼 | 최대 길이 |
|--------|----------|
| YouTube Shorts | 60초 |
| Instagram Reels | 90초 |
| TikTok | 10분 (권장 15-60초) |

### 파일 형식

| 항목 | 권장 값 |
|------|---------|
| 컨테이너 | MP4 |
| 비디오 코덱 | H.264 |
| 오디오 코덱 | AAC |
| 프레임레이트 | 30fps (또는 60fps) |
| 비트레이트 (비디오) | 4-8 Mbps |
| 비트레이트 (오디오) | 128-256 kbps |

## 2. 16:9 → 9:16 변환

### 크롭 방식

```
원본 (16:9):
┌────────────────────────────────────┐
│                                    │
│    ┌──────────┐                    │
│    │  크롭    │                    │
│    │  영역    │                    │
│    │  (9:16)  │                    │
│    └──────────┘                    │
│                                    │
└────────────────────────────────────┘

결과 (9:16):
┌──────────┐
│          │
│  크롭된  │
│  영역    │
│          │
└──────────┘
```

### 크롭 위치 옵션

```
center (기본): 중앙 기준 크롭
left: 왼쪽 기준 크롭 (화자가 왼쪽에 있을 때)
right: 오른쪽 기준 크롭 (화자가 오른쪽에 있을 때)
```

### 화면 분할 방식

```
상단-하단 분할:
┌──────────┐
│  원본    │
│  영상    │
├──────────┤
│  자막/   │
│  그래픽  │
└──────────┘

좌우 분할 (리액션 영상):
┌─────┬─────┐
│     │     │
│원본 │리액션│
│     │     │
└─────┴─────┘
```

## 3. 자막 스펙

### 위치 및 크기

```
┌──────────────────┐
│                  │
│                  │
│                  │
│                  │
│   ┌──────────┐   │
│   │   자막   │   │ ← 하단에서 10-15%
│   └──────────┘   │
│                  │
└──────────────────┘
```

### 스타일 가이드

| 항목 | 권장 값 |
|------|---------|
| 폰트 | 굵은 산세리프 (Bold Sans-serif) |
| 크기 | 화면 너비의 70-80% |
| 색상 | 흰색 (#FFFFFF) |
| 테두리 | 검정 2px |
| 배경 | 반투명 검정 (선택) |
| 위치 | 하단 중앙, 여백 50-100px |

### ffmpeg 자막 필터

```bash
# 기본 자막
subtitles=subtitle.srt

# 스타일 적용
subtitles=subtitle.srt:force_style='FontSize=24,FontName=Arial,Bold=1,PrimaryColour=&HFFFFFF,OutlineColour=&H000000,Outline=2,Shadow=1,Alignment=2,MarginV=50'
```

### 자막 타이밍

```
✅ 좋은 타이밍:
- 말하는 속도와 동기화
- 문장 단위로 끊기
- 1줄당 2-3초

❌ 피해야 할 것:
- 너무 빠른 전환 (1초 미만)
- 너무 긴 자막 (5초 이상)
- 말과 어긋나는 타이밍
```

## 4. 플랫폼별 특이사항

### YouTube Shorts

```
- 최대 60초
- 세로 영상 자동 인식 (#Shorts 태그 불필요)
- 자체 음악 라이브러리 사용 가능
- 1080x1920 권장
```

### Instagram Reels

```
- 최대 90초
- 커버 이미지 별도 설정 가능
- 쇼핑 태그 지원
- 1080x1920 필수
```

### TikTok

```
- 최대 10분 (권장 15-60초)
- 듀엣/스티치 기능 고려
- 세로 + 음악 조합이 핵심
- 다양한 해상도 지원
```

## 5. 화질 최적화

### 인코딩 설정 (ffmpeg)

```bash
# 고화질 (권장)
-c:v libx264 -preset medium -crf 23

# 빠른 인코딩
-c:v libx264 -preset fast -crf 25

# 최고 화질
-c:v libx264 -preset slow -crf 18
```

### CRF 가이드

| CRF 값 | 화질 | 파일 크기 |
|--------|------|----------|
| 18 | 거의 무손실 | 매우 큼 |
| 23 | 높음 (권장) | 적당함 |
| 28 | 중간 | 작음 |
| 35 | 낮음 | 매우 작음 |

## 6. 클립 선정 가이드

### 좋은 쇼츠 구간

```
✅ 적합한 구간:
- 강렬한 후킹으로 시작
- 하나의 완결된 메시지
- 말이 자연스럽게 끊기는 지점
- 시각적으로 흥미로운 장면
- 감정적 반응이 있는 순간

❌ 피해야 할 구간:
- 중간에 말이 끊기는 곳
- 긴 침묵이나 어색한 순간
- 화면 전환이 어색한 지점
- 저작권 음악이 큰 부분
- 맥락 없이는 이해 안 되는 내용
```

### 타임스탬프 선정 팁

```
1. 자막에서 핵심 키워드 검색
2. 해당 구간 앞뒤 5초 확인
3. 말이 시작되는 지점 찾기
4. 60초 이내로 끊기 좋은 지점 확인
5. 시작과 끝이 자연스러운지 확인
```

## 7. 체크리스트

### 제작 전

```
□ 원본 영상 해상도 확인
□ 클립 구간 선정 (대본 참조)
□ 자막 파일 준비 (SRT)
□ 크롭 위치 결정
```

### 제작 후

```
□ 비율 확인 (9:16)
□ 해상도 확인 (1080x1920)
□ 길이 확인 (60초 이내)
□ 자막 가독성 확인
□ 오디오 레벨 확인
□ 시작/끝 전환 자연스러움
□ 파일 크기 적정 (100MB 이내 권장)
```

## 8. 트러블슈팅

### 일반적인 문제

```
문제: 화질이 낮음
→ 원본 화질 확인, CRF 값 낮추기 (18-20)

문제: 자막이 잘 안 보임
→ 폰트 크기/테두리 조정, 배경 추가

문제: 크롭 후 중요 부분이 잘림
→ 크롭 위치 변경 (center → left/right)

문제: 파일 크기가 너무 큼
→ CRF 값 높이기, 비트레이트 낮추기

문제: 업로드 후 화질 저하
→ 더 높은 비트레이트로 인코딩
```
